## 需求

* 需要```pillow```包支持
* Python 3.x

## 使用说明

调用以下代码：

```python
from convert_points_image import img

try1 = img('./images/123.jpg')
try1.RGB_to_points(40, 40, converse=True, save_filename='./images/convert1.txt')
```
